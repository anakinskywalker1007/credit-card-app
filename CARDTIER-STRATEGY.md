# OpenClaw — CardTier.com Growth, SEO & Monetization Instructions

### Based on Live Site Audit — March 2026

-----

## SITE OVERVIEW (What Exists Today)

CardTier.com is currently a single-page application featuring:

- 50 credit cards ranked by "real net value" (signup bonuses, annual credits, fees factored in)
- Category filters: Travel, Cash Back, Business, Premium, No Annual Fee
- Sort options: Value High/Low, Bonus Size, Annual Fee, Name A-Z
- A side-by-side card comparison tool
- Individual card modals showing: Sign-Up Bonus, Annual Fee, Credit Score, APR, Value Breakdown, Rewards, Perks, Transfer Partners

What is missing entirely:

- Individual card pages with URLs (no indexable subpages)
- Any written editorial content or blog
- Any SEO metadata beyond the homepage title
- An email capture or lead magnet
- Any monetization layer (ads, affiliate links, affiliate network integration)
- An About page or trust signals
- A beginner-oriented entry point or quiz
- Any social media presence

The core problem: The entire site is one URL. Google cannot index 50 individual card rankings because they live inside a JavaScript app with no separate pages. This means the site has essentially zero SEO surface area right now. This is the single highest-priority technical issue.

-----

## YOUR MANDATE

Run four tracks simultaneously and report on all four every session. Do not treat these as phases — execute them in parallel. Prioritize ruthlessly by impact.

-----

## TRACK 1 — CRITICAL TECHNICAL FIXES (Do First)

### Fix 1: Create Individual Card Pages (Highest Priority)

The site must generate a unique, indexable URL for every card.

What needs to exist:
- cardtier.com/cards/chase-sapphire-preferred
- cardtier.com/cards/amex-gold
- cardtier.com/cards/capital-one-venture-x
- … one for all 50 cards

Each page must contain:
- The card's full data expanded into written prose
- A detailed written review (300–600 words minimum per card)
- Who the card is best for (beginner / optimizer / traveler)
- Clear pros and cons section
- The CardTier net value score with calculation explanation
- An "Apply Now" button (placeholder for now, affiliate link later)
- Internal links to related cards and comparison pages

### Fix 2: Create Category Landing Pages

- cardtier.com/best-travel-credit-cards
- cardtier.com/best-cash-back-credit-cards
- cardtier.com/best-credit-cards-no-annual-fee
- cardtier.com/best-business-credit-cards
- cardtier.com/best-premium-credit-cards
- cardtier.com/best-credit-cards-for-beginners

Each with 200–300 word editorial intro, top 5–8 cards, "Last Updated: [Month Year]"

### Fix 3: Add a Beginner Entry Point

- cardtier.com/start — quiz (credit score, spending category, fee comfort, goal)
- "New to credit cards?" banner on homepage
- cardtier.com/guides/how-to-get-your-first-credit-card

### Fix 4: Technical SEO Baseline

- Unique title tags: "[Card Name] Review 2026 — CardTier"
- Meta descriptions under 155 chars
- H1 tags matching search intent
- Schema markup: Review + Product schema (enables star ratings in Google)
- Sitemap.xml submitted to Google Search Console
- PageSpeed Insights: flag anything under 80 mobile
